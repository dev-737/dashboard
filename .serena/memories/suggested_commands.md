# Suggested Commands

## Development
*   `npm run dev`: Start the development server.
*   `npm run build`: Build the application for production.
*   `npm run start`: Start the production server.

## Code Quality
*   `npm run lint`: Run Biome linter.
*   `npm run format`: Format code with Biome.
*   `npm run typecheck`: Run TypeScript compiler check.

## Database (Prisma)
*   `npx prisma generate`: Generate Prisma Client.
*   `npx prisma db push`: Push schema changes to the database (prototyping).
*   `npx prisma migrate dev`: Create and apply migrations (production-ready changes).
*   `npx prisma studio`: Open Prisma Studio to view data.

## Utilities
*   `npx biome check --apply .`: Fix linting issues automatically where possible.
